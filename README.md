## ⌨️Development
```
git clone git@github.com:robinroy03/videoeditor.git
cd videoeditor
pnpm i
```

Start the frontend
```
pnpm run dev
```

Start the backend
```
pnpm dlx tsx app/videorender/videorender.ts
```
